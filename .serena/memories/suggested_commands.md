# Suggested Commands

## Build & Test
- `npm run build` — Build all packages (turbo)
- `npm run test` — Run all tests (turbo + vitest)
- `npm run lint` — Lint all files (biome check .)
- `npm run format` — Format all files (biome format --write .)
- `npm run typecheck` — Type check all packages

## Development
- `npm run dev` — Start dev servers
- `npm run test:integration` — Run integration tests

## Release
- `npx changeset` — Create changeset
- `npm run release` — Version + publish

## System (macOS/Darwin)
- `git`, `ls`, `cd`, `grep`, `find` — standard unix utils
